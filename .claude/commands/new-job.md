Generate a recruiter prep doc for a scheduled recruiter call.

Goal: produce `recruiter-prep.md` in the job folder so you walk into the call prepared.

**When to use:** When a recruiter call is scheduled — either for a cold-apply role (folder already exists with `jd.md` and `resume.md`) or an inbound role (no prior folder).

**Cold-apply path** (folder exists): Read `jd.md` and `resume.md` from the existing job folder, then run `SKILL-recruiter-prep.md`.

**Inbound path** (no prior folder): Ask for the job description and URL. Save the JD as `jd.md` in a new folder using the naming convention `jobs/[company]-[role]-[YYYY-MM-DD]/`. Then run `SKILL-recruiter-prep.md`.

Filename convention: `jobs/[company]-[role]-[YYYY-MM-DD]/` using today's date.

Full job file (analysis, Why This Job, interview prep) is only created if you advance past the recruiter screen.
